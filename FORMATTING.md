# Cell formatting

Font: family, size, bold, italic, underline, strikethrough, color.
Fill: solid color only; no gradient/pattern.
Borders: each side; solid/dashed/dotted/double; color.
Alignment: horizontal, vertical, wrapping, rotation (0-90 degrees, stacked).
Number formats: excel compatible -- not really formatting, moreso need to provide strings
Conditional formatting: just programmatically controlling the above.
Data validation: dropdowns, checkbox rendering.
Column/row size, hidden rows/columns, grouping.
Frozen rows/columns.
